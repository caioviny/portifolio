GeoSup

O GeoSup é uma aplicação web desenvolvida em PowerApps para o setor de Geotecnologia, com foco na organização de demandas. Utiliza o modelo Kanban para movimentação entre status, armazena o histórico das tarefas, permite criar, editar e atribuir demandas a usuários específicos. Existem dois tipos de acesso: líderes de processo, que veem todas as demandas, e usuários comuns, que veem apenas as suas. Com essas funcionalidades, o GeoSup oferece uma solução personalizada, moderna e eficiente para a gestão de atividades internas do setor.
